En el primer ejercicio de base de datos la query correcta es la D

Select P.Name
from Projects P
where exists ( select 1
    from SPP, Suppliers S
    where SPP.ProjectId = P.ProjectId
    and SPP.SupplierId = S.SupplierId
    and S.city = P.city )
